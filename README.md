# flask-search

## Download Project
"git clone https://github.com/mohsinenur/flask-search.git" from your cmd

## Prerequsite
You have to installed Python 3 and Virtualenv in your OS.

## Setup
1. Go to the project folder by "cd flask-search"
2. Setup virtualenv by "virtualenv venv"
3. Activate virtualenv by "source venv/bin/activate" (this works if your OS is linux)
4. Now install requirements by "pip install -r requirements.txt"

## Run Project
If previous command worked well then,
"python run.py" this command will run the app.
